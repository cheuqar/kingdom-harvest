# 天國大富翁 Kingdom Harvest - 專案完成報告

我已經為您建立並實作了完整的「天國大富翁」網頁遊戲，並根據您的需求進行了優化。

## 1. 專案概覽

*   **技術棧**: React + Vite (單頁應用程式)
*   **風格**: 現代化深色主題 (Dark Mode)，搭配玻璃擬態 (Glassmorphism) 與動態效果。
*   **核心機制**:
    *   **土地買賣**: 包含聖經地名、系列加成、旅店建造。
    *   **事件卡**: 77 張完整事件卡，涵蓋收租、播種、試煉、神蹟。
    *   **種子機制**: 實作了「多種的多收」邏輯，種子數量影響收益。
    *   **神蹟卡**: 可收集並在關鍵時刻使用（如破產前保命）。
    *   **聖經問答**: 購買土地前需回答問題，答錯進入拍賣池。
    *   **破產與勝利**: 自動偵測破產，最後倖存者獲勝。

## 2. 最新更新 (Pacing & UI)

*   **資產檢視 UI**: 點擊左側隊伍卡片，可開啟「資產詳情」視窗，查看：
    *   現金、種子、神蹟卡數量。
    *   擁有的土地列表（按系列分組），顯示當前旅店數與租金。
    *   擁有的神蹟卡詳情。
*   **遊戲節奏優化 (20分鐘局)**:
    *   **起始資金**: 調整為 **$2500** (原 $5000)，加速經濟流動與破產壓力。
    *   **擲骰機率**: 
        *   1-3: 抽土地卡 (50% 機率，大幅增加土地獲取速度)。
        *   4: 建旅店 (若無土地則改為抽土地卡)。
        *   5-6: 事件卡。
    *   **系列加成**: 提升至 **2倍 / 3倍 / 4倍** (原 1.5/2/3)，獎勵收集成套土地。

## 3. 檔案結構

*   `src/config/`: 包含 `config.json`, `lands.json`, `events.json`, `questions.json`，方便後續調整數值與內容。
*   `src/engine/`: `effects.js` 包含所有卡牌效果邏輯。
*   `src/hooks/`: `useGameEngine.js` 處理遊戲核心迴圈。
*   `src/state/`: `GameContext.jsx` 管理全域狀態 (Redux-like pattern)。
*   `src/components/`: 各個 UI 元件 (`GameBoard`, `MainArea`, `CardDisplay`, `TeamList`, `TeamAssetsModal` 等)。

## 4. 如何開始

1.  確保終端機在 `app` 目錄下：`cd app`
2.  安裝依賴 (已完成): `npm install`
3.  啟動開發伺服器 (已執行): `npm run dev`
4.  打開瀏覽器訪問: `http://localhost:5174` (或終端機顯示的埠號)

## 5. 遊戲操作指南

*   **開始遊戲**: 設定隊伍數量與名稱。
*   **查看資產**: 隨時點擊左側隊伍卡片查看詳細資產。
*   **回合流程**:
    *   **擲骰**: 點擊按鈕擲骰。
    *   **1-3 (土地)**: 抽土地卡 -> 回答問題 -> 購買或放棄 (放棄則進入拍賣池)。
    *   **4 (旅店)**: 若有土地，可選擇一塊地建旅店；若無土地，自動轉為抽土地卡。
    *   **5-6 (事件)**: 抽事件卡 -> 自動執行效果 (或獲得神蹟卡)。
*   **神蹟卡**: 獲得後會顯示在下方「我的神蹟卡」區域，點擊「使用」即可觸發效果。
*   **結束回合**: 操作完成後點擊結束，輪到下一隊。
